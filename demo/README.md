# Settings
- Press 'F9', click 'Load XML Decals', and select 'images/1-2250.xml' (1:2250 image scale)
- In Netedit, press 'F10' and ensure 'lefthand' is true to simulate UK road system

# Network Guide
Lanes names:
- (Street Name)\_(Lane Number)\_(Segment Number)

Junction names: 
- EJ\_(X) -> End Junction X
- CJ\_(Y) -> Connecting Junction Y
