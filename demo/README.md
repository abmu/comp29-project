# Settings
- Press 'F9' and load XML decals called "decals.xml" to import background "1-2250.png" (means 1:2250 image scale)
- Press 'F10' and switch lefthand to True to simulate UK road system

# Network Guide
Lanes names:
- (Street Name)_(Lane Number)_(Segment Number)

Junction names: 
- EJ_(X) -> End Junction X
- CJ_(Y) -> Connecting Junction Y
